البيان | Al-Bayan
> تطبيق إسلامي شامل لقراءة القرآن الكريم والحفظ ومواقيت الصلاة والأذكار
## الميزات الرئيسية
### القرآن الكريم
- المصحف الشريف الكامل مع خط واضح
- ٢٠+ قارئ مع إمكانية التحميل
- التفاسير المعتمدة (ابن كثير، الجلالين، الميسر)
- أحكام التجويد
- العلامات المرجعية والبحث
### نظام الحفظ الذكي
- خطط حفظ مرنة (٥ أشهر - سنة)
- نظام التكرار المتباعد (SRS)
- اختبارات ذاتية وتتبع التقدم
- تقويم الإنجازات ومشاركتها
### مواقيت الصلاة والأذان
- حساب دقيق بـ ١٢ طريقة معتمدة
- ١٣+ مؤذن مع إمكانية التخصيص لكل صلاة
- تنبيهات ذكية قبل وبعد الصلاة
- تحديد اتجاه القبلة
### الأذكار والأحاديث
- مكتبة أذكار شاملة مصنفة
- عداد تسبيح رقمي
- الكتب الستة للأحاديث مع البحث
### المزيد
- التقويم الهجري مع المناسبات الإسلامية
- اذاعات القرآن الكريم و مشايخ التلاوة
- دليل الحج والعمرة
- الوضع الليلي
- يعمل بدون إنترنت
## التقنيات
- React + TypeScript
- Capacitor (Android)
- TailwindCSS
- مكتبة adhan-js لحساب المواقيت
